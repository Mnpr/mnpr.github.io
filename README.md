# sudesh.io

- [Nuxt3](https://nuxt.com)

## Setup

install dependencies:

```bash
npm install
||
pnpm install
||
yarn install
```

## Development Server

@`http://localhost:3000`:

```bash
npm run dev
||
pnpm run dev
||
yarn dev
```

## Production


```bash
npm run build
||
pnpm run build
||
yarn build
```

Local production build preview:

```bash
npm run preview
||
pnpm run preview
||
yarn preview
```

more info @[deployment documentation](https://nuxt.com/docs/getting-started/deployment).
